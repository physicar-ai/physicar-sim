# physicar-sim

Gazebo Harmonic simulation assets and runtime helpers for **PhysiCar**.

Bundles the gzweb 3D viewer (`share/`), the websocket launch file
(`websocket.gzlaunch`), an HTTP track-management API (`sim_api.py`), and
an installer/service script (`sim.sh`).
